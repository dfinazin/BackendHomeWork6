1. npm init - Инициализация проекта (с ключом -y без уточняющих вопросов)
2. В package.json добавить строку ' "type": "module" ' - Для использования 'import <> from <>' вместо '<> = require(<>)'
    В таком случае переменные путей необходимо определить как 
        import {fileURLToPath} from "url"
        const __filename = fileURLToPath(import.meta.url)
        const __dirname = path.dirname(__filename)
